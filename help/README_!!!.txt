Инструкция:
1. Открываем firefox (в других браузерах она работает неправильно (пока что))
2. Сервак питоновский запускать как обычно из корня: python manage.py runserver дальше сюды: http://127.0.0.1:8000/
3. Жмем "Start Game"
4. Играем, для возврата на главную жмем "Змейка-Фруктоедка"

Snake_v6_json - сохранение данных с помощью json
Snake_v7_models - сохранение данных с помощью моделей питона
